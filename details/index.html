Details

Find out more about the best Twitter app for iOS.

NeoFreeBird is a modified version of the X/Twitter app. It reverts branding, unlocks premium features, removes ads, disables data-collection, and is in general a vision of how Twitter would've been had Elon Musk not bought it.

Whilst you can use the original X app, other modded apps, or the last Twitter-branded version, we believe NeoFreeBird stands out. We keep NeoFreeBird up to date with new Twitter features and constantly refine our rebranding, something other modded clients don't do. And compared to using the last Twitter branded build, NeoFreeBird isn't affected by API changes and X can't detect if you're using NeoFreeBird.

The only thing NeoFreeBird does is to replace image assets and language strings, modify feature flags, and incorporate a modified BHTwitter fork, all of which is available on GitHub, meaning you can see exactly what makes NeoFreeBird tick.

Due to there only being 3 people working on NeoFreeBird, and all of us having full-time jobs or school to attend to, we can't provide any form of ETAs on new releases.
